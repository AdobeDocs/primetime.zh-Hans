# 如何使用功能管理器{#how-to-use-feature-managers}

有两种方法可以使用功能管理器启用、禁用和配置功能：

* 您可以通过将配置信息从配置文件传递到媒体播放器来创建和配置功能管理器。
* 您可以使用管理器工厂启用和禁用功能，无需遍历代码。
