# 不插入广告的DVR {#dvr-without-ad-insertion}

默认情况下，支持DVR的实时流会公开查看器可以暂停和搜寻的可搜索范围。

您可以使用以下附加功能增强DVR的播放器控制栏：

* 在控制栏两侧隐藏当前播放时间戳和剩余时间戳。
* 突出显示控制栏中的可搜索范围。
* 将控制栏的持续时间更改为流的可搜索范围，而不是流的播放范围。

   可搜索范围是可播放范围的子集，广告可以安全插入该可播放范围。
* 选择输入流的自定义起点。
