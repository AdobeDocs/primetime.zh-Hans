# 未插入广告的DVR {#dvr-without-ad-insertion}

默认情况下，具有DVR支持的实时流显示可查看的范围，查看器可在该范围中暂停和搜索。

您可以使用以下附加功能增强DVR的播放器控制栏：

* 隐藏控件栏两侧的当前播放时间戳和剩余时间戳。
* 高亮显示控制栏中的可搜索范围。
* 将控制栏的持续时间更改为流的可搜索范围，而不是流的播放范围。

   可搜索范围是可安全插入广告的可播放范围的子集。
* 选择何时进入流的自定义起始点。
