# 带有广告插入的DVR {#dvr-with-ad-insertion}

默认情况下，支持DVR的实时流会公开查看器可以暂停和搜寻的可搜索范围。

广告时间在控制栏中显示为红色条，如下所示：

<!--<a id="fig_720DD22D2318485EAB4BEA55C30D5ECF"></a>-->

![](assets/dvr-with-ads.jpg)

可搜索范围为灰色，红点表示广告时间。
