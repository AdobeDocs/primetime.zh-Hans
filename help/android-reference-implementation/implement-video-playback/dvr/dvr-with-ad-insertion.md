# 广告插入{#dvr-with-ad-insertion}的DVR

默认情况下，具有DVR支持的实时流显示可查看的范围，查看器可在该范围中暂停和搜索。

广告断点在控制栏中显示为红条，如下所示：

<!--<a id="fig_720DD22D2318485EAB4BEA55C30D5ECF"></a>-->

![](assets/dvr-with-ads.jpg)

可搜索范围为灰色，红点表示广告分段。
