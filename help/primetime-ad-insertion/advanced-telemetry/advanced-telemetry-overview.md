# 高级遥测 {#advanced-telemetry}

客户可以使用PrimetimeAd Insertion捕获与广告和会话相关的事件和量度，以揭示内容和广告错误的主要原因，从而深入了解视频和广告性能。  广告请求可以在原始请求级别捕获，从而使操作员能够调试直接来自提供商的特定VAST或VMAP标记。
