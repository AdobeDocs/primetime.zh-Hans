# 报表API {#report-api}

用户界面为客户和启用(Adobe)团队管理了角色。 客户可以访问门户，然后可以创建和编辑其配置。 用户还可以在用户界面上查看其广告展示次数的报表。

API在后台运行，以便客户和管理员能够与后端基础架构进行通信。

请参阅 [适用于Auditude的报表API](../assets/auditude-report-api.pdf).
