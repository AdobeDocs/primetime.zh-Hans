# 对VOD {#changes-to-vod}的更改

适合VOD内容之一播放的广告时间线可能不适合后续播放。 您可以替换VOD时间轴而不更改内容。

您可能希望替换VOD时间轴的情况包括：

* 替换本地广告，但在C3窗口中保留国家广告。
* 在C3窗口关闭后替换烧入广告。
* 将旧的C3广告动态替换为持续时间更长的较新广告。
* 插入更少或完全没有广告。

您可以使用原始M3U8文件和`pttimeline`查询参数的不同值提交新的广告插入请求来替换广告时间线。
