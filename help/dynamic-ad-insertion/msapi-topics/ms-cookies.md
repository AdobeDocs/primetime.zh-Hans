# Cookie {#cookies}

Primetime广告插入使用cookies进行内部用途，并便于使用流和广告服务器。

## Cookie列表

Primetime广告插入可维护以下Cookie:

* `ssaiPub` 帮助从需要cookie身份验证的流中检索内容URL以获取文 `m3u8` 件。
* `ssai3` 保存需要的广告服务器的会话Cookie。
* `AWSELB` 并保 `ssaiSession` 留内部信息。
