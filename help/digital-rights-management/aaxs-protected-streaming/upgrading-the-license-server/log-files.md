# 日志文件{#log-files}

Adobe Access Server for Protected Streaming应用程序生成的日志文件将位于指定的目录中 *LicenseServer.LogRoot*.

>[!NOTE]
>
>如果在服务器运行时删除或移动了当前日志文件，则可能无法重新创建日志文件，并且某些日志信息将丢失。
