# 日志文件{#log-files}

Adobe Access Server为受保护的流应用程序生成的日志文件将位于 *LicenseServer.LogRoot指定的目录中*。

>[!NOTE] {class=&quot;- topic/note &quot;}
>
>如果在服务器运行时删除或移动当前日志文件，则可能不会重新创建日志文件，并且某些日志信息将丢失。
