# 租户日志文件{#tenant-log-file}

每个租户的租户日志文件， [!DNL flashaccess-tenant.log]，位于 *LicenseServer.LogRoot* [!DNL /flashaccesserver/tenants/]*租户名称*. 租户日志包含描述为此租户生成的每个许可证的审核信息。
