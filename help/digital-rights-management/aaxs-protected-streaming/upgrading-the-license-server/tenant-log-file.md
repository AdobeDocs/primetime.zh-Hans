# 租户日志文件{#tenant-log-file}

每个租户的租户日志文件[!DNL flashaccess-tenant.log]位于&#x200B;*LicenseServer.LogRoot* [!DNL /flashaccesserver/tenants/]*tenantname*&#x200B;中。 租户日志包含审计信息，用于描述为此租户生成的每个许可证。
