# 更新全局配置文件{#updating-the-global-configuration-file}

可以随时修改[!DNL flashaccess-global.xml]中的HSM密码，这些更改将在服务器下次重新加载配置文件时生效。 但是，不会重新加载对“记录”和“缓存”元素所做的更改；这些元素中的任何更改都需要重新启动服务器。
