# 更新全局配置文件{#updating-the-global-configuration-file}

中的HSM密码可 [!DNL flashaccess-global.xml] 以随时修改，更改将在服务器下次重新加载配置文件时生效。 但是，不会重新加载对“记录”和“缓存”元素的更改；这些元素中的任何更改都需要重新启动服务器。
