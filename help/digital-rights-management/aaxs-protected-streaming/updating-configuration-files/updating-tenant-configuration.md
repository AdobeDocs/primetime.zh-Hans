# 更新租户配置文件{#updating-the-tenant-configuration-file}

中指定的所有值 [!DNL flashaccess-tenant.xml] 可以随时修改，这些更改将在下次服务器重新加载配置文件时生效。 此外，服务器还会检查租户配置文件中引用的所有凭据(.pfx)文件和Packager允许列表证书文件中的更改。
