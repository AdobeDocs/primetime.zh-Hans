# 更新租户配置文件{#updating-the-tenant-configuration-file}

可以随时修改[!DNL flashaccess-tenant.xml]中指定的所有值，这些更改将在服务器下次重新加载配置文件时生效。 此外，服务器检查租户配置文件中引用的所有凭据(.pfx)文件和打包程序允许列表证书文件中的更改。
