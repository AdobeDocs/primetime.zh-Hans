# 要求 {#requirements}

您需要确保已安装以下软件：

* Tomcat 6.0或更高版本
* MySQL等数据库（仅许可证服务器组件需要）
* Java 1.6或更高版本
* Ant 1.8或更高版本（使用示例构建脚本）

安装Tomcat和MySQL后，请从Adobe获取凭据。
