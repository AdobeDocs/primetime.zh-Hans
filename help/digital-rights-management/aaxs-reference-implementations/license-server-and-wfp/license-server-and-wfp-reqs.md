# 要求 {#requirements}

您需要确保安装了以下软件：

* Tomcat 6.0或更高版本
* 诸如MySQL的数据库（仅对于License Server组件需要）
* Java 1.6或更高版本
* Ant 1.8或更高版本（使用示例生成脚本）

安装Tomcat和MySQL后，请从Adobe获取凭据。
