# 创建新策略{#create-a-new-policy}

在打包任何内容之前，必须创建一个或多个策略。 有关策略中可能指定的使用规则的概述，请参阅Adobe Access SDK概述中的使用规则。

要创建新策略，请单击“新建”并输入策略名称。 填写所需的策略属性（所有设置都是可选的）。 完成后，单击&#x200B;**[!UICONTROL Save]**。 策略将保存为资源目录中的[!DNL policyname.pol]。
