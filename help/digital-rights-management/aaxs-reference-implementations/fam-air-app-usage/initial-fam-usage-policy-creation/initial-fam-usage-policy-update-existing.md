# 更新现有策略 {#update-an-existing-policy}

要更新现有策略，请从下拉列表中选择文件名并单击 **[!UICONTROL Open]**. 修改任何所需的策略属性。 除了与身份验证和许可证链接相关的属性外，可以修改所有属性。

完成后，单击 **[!UICONTROL Save]**. 中的策略文件 [!DNL Resource] 目录将被更新版本替换。

>[!NOTE]
>
>即使策略名称已更改，文件名称在 [!DNL Resource] 将不会修改目录。
