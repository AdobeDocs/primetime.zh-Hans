# 自定义数据 {#custom-data}

下表介绍了“自定义数据”首选项：

| 首选项 | 描述 |
|---|---|
| 自定义策略属性 | 指定许可证服务器在颁发许可证时可以使用的自定义属性。 |
| 自定义许可证属性 | 指定自定义属性，这些属性将显示在颁发给客户端的许可证中。 客户端应用程序将有权访问这些属性。 |
