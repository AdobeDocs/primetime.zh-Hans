# 构建Packager Server和AIR应用程序 {#building-the-packager-server-and-air-application}

使用Flash Access Manager需要两个组件：

* Flash Access Manager AIR应用程序
* Packager Server( [!DNL flashaccess-packager.war])

这两个组件都以源和二进制形式分发，并带有引用实现。
