# 策略更新列表首选项 {#policy-update-list-preferences}

| 首选项 | 说明 |
|---|---|
| 许可证服务器凭据 | Adobe颁发的License Server凭证。 此凭据用于签署策略更新列表。 |
| 文件名 | 包含证书和私钥 [!DNL .pfx]的PKCS#12()文件。 文件必须位于“目录” [!DNL Resource] 中。 |
| 文件口令 | 文件的口 [!DNL .pfx] 令 |
