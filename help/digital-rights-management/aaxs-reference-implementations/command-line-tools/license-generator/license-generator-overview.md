# 许可证生成器概述{#license-generator-overview}

使用License Generator命令行工具，您无需客户端向服务器发送许可证请求即可生成许可证。 预生成的许可证可以嵌入到内容中，或通过其他机制（如简单的HTTP Web服务器）交付到客户端。
