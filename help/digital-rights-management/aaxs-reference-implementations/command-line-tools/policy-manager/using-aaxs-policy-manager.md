# 使用Policy Manager {#using-policy-manager}

使用Policy Manager ，您可以创建和管理策略。 运行Policy Manager之前，可以选择在配置文件中设置Policy Manager属性的值。 配置文件指定将应用于所有策略的信息。 也可以在命令行上指定所有Policy Manager属性。
