# 使用策略管理器 {#using-policy-manager}

使用Policy Manager ，您可以创建和管理策略。 在运行策略管理器之前，可以选择在配置文件中设置策略管理器属性的值。 配置文件指定将应用于所有策略的信息。 也可以在命令行上指定所有Policy Manager属性。
