# 下载到拥有 {#download-to-own}

使用下载后拥有(DTO)使用模式，用户可以下载内容以供在线或离线使用，并获得该内容的永久许可证。 在请求许可证时，用户必须进行身份验证，以便服务器可以验证用户是否已购买内容。
