# Download-To-Own {#download-to-own}

使用“下载到自有”(DTO)使用模式，用户可以下载内容供在线或离线使用，并获得内容的永久许可证。 在请求许可证时，用户必须进行身份验证，这样服务器才能验证用户是否已购买该内容。
