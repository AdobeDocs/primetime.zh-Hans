# 订阅{#subscription}

一些服务优惠付费的订阅，用户只要继续支付月费，就可以无限制地访问大型内容库。 许可证服务器为每个内容颁发唯一的许可证，并颁发根许可证，其过期时间与订阅时间相符。 每月，当用户重新发布其订阅时，根许可证也可以续订。 在请求许可证时，用户需要进行身份验证，以便服务器可以验证用户的订阅是否为最新。
