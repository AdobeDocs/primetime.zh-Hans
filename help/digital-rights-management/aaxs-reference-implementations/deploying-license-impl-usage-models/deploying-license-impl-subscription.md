# 订阅 {#subscription}

某些服务提供付费订阅，只要用户继续支付月费，用户就可以无限制地访问大型内容库。 许可证服务器为每个内容发出唯一的许可证，并发出一个根许可证，其到期时间与订阅期相同。 每月，当用户续订其订阅时，也可以续订根许可证。 在请求许可证时，用户需要进行身份验证，以便服务器可以验证用户的订阅是否为最新状态。
