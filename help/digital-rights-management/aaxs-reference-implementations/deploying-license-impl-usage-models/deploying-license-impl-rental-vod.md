# 租赁/视频点播 {#rental-video-on-demand}

使用视频点播(VOD)使用模型，提供包含基于时间的限制的内容。 例如，用户可以选择在30天内播放内容；但是，一旦播放开始，用户最多有48小时完成观看，之后内容将无法播放。 在请求许可证时，用户必须进行身份验证，以便服务器可以验证用户是否具有租赁帐户。
