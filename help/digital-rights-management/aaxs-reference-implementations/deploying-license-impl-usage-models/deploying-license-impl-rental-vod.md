# 租赁/视频点播{#rental-video-on-demand}

通过视频点播(VOD)使用模型，内容提供时间限制。 例如，用户可以选择在30天内播放内容；但是，一旦播放开始，用户最多有48小时的时间才能完成观看，此后内容将不再可播放。 在请求许可证时，用户必须进行身份验证，以便服务器可以验证用户是否具有租赁帐户。
