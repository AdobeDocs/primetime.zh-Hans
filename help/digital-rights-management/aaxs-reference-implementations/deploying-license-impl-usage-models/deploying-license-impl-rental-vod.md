# 租赁／视频点播 {#rental-video-on-demand}

通过视频点播(VOD)使用模型，内容提供时间限制。 例如，用户可以在30天内播放内容；但是，一旦播放开始，用户最多有48小时的时间可以完成观看，之后内容将不再可播放。 在请求许可证时，用户必须进行身份验证，这样服务器才能验证用户是否拥有租赁帐户。
