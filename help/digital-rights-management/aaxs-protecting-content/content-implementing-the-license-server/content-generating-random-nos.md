# 生成随机数{#generating-random-numbers}

硬件随机数生成器可用于Linux服务器，以确保生成足够的熵。 如果计算机无法生成足够的熵，则需要随机源的Adobe访问操作将在等待`/dev/random`的数据时阻塞。
