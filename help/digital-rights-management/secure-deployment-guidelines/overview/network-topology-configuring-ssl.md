# 配置SSL {#configuring-ssl}

仅当需要支持Flash媒体Rights Management服务器1.x客户端时，才需要安全套接字层(SSL)。

Adobe PrimetimeDRM密钥服务器需要使用客户端身份验证的SSL。 有关详细信息，请参阅[使用Adobe PrimetimeDRM密钥服务器](../../using-the-drm-key-server/requirements.md)。
