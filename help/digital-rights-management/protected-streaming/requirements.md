# 要求{#requirements}

您的构建环境必须满足以下最低要求：

* Microsoft Windows Server 2008或Red Hat® Enterprise Linux® 5.6
* Oracle Java JRE 1.6(创建自定义授权扩展需要Oracle Java JDK 1.6)
* 位于DVD的[!DNL Third Party\Tomcat\6.0.18]文件夹中的Apache Tomcat® 6
* Adobe颁发的凭据
