# 概述 {#using-the-drmstatusevent-class-overview}

当 `DRMStatusEvent` 受Primetime DRM保护的内容开始成功播放时，将调度一个对象。 （成功意味着许可证已经过验证，并且用户已通过身份验证并获得查看内容的授权）。

该对 `DRMStatusEvent` 象包含与许可证相关的信息，包括许可证是否可以脱机使用或许可证过期以及内容是否无法再查看。
