# 正在覆盖策略选项{#overriding-policy-options}

发布许可证时，许可证服务器可能会重写策略中指定的使用规则。 如果策略指定了开始日期，则不会在该开始日期之前生成许可证。 但是，在生成许可证后，可以在许可证中设置未来开始日期。 此选项应谨慎使用，因为客户端不会阻止用户将系统时间向前移动以规避开始日期。
