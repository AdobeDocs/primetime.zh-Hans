# 正在覆盖策略选项{#overriding-policy-options}

在颁发许可证时，许可证服务器可以覆盖策略中指定的使用规则。 如果策略指定了开始日期，则不会在该开始日期之前生成许可证。 但是，可以在生成许可证后设置许可证中的未来开始日期。 应谨慎使用此选项，因为客户端不会阻止用户将系统向前移动以规避开始日期。
