# 配置SSL {#configuring-ssl}

只有在需要支持Flash Media Rights Management Server 1.x客户端时，才需要安全套接字层(SSL)。 Adobe访问密钥服务器需要使用客户端身份验证的SSL。 有关详细信息，请参阅*使用Adobe访问密钥服务器*。
