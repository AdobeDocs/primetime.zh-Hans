# 命令行工具要求 {#command-line-tools-requirements}

* Java 1.5或更高版本。
* 由Adobe颁发的包装程序、传输和许可证服务器凭据（证书和密码）。

   这些凭据用于加密和签署视频文件、签署策略更新和撤销列表以及预生成许可证。

>[!NOTE]
>
>由于Java错误，您在命令行中键入的任何参数（如文件名、DRM策略名称或说明）只能使用操作系统默认字符集中的字符。
