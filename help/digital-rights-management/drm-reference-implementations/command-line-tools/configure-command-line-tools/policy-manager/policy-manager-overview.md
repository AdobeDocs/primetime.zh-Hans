# 概述 {#overview}

使用Primetime DRM Policy Manager命令行工具( [!DNL AdobePolicyManager.jar])，以创建和管理DRM策略。

运行之前 [!DNL AdobePolicyManager.jar]，您必须在配置文件的Policy Manager Properties部分设置属性。 在配置文件中设置的值将应用于所有DRM策略。

>[!NOTE]
>
>您还可以从命令行指定所有Policy Manager属性。
