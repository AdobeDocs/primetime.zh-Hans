# 相对结束日期{#relative-end-date}

相对结束日期指定了许可证到期日期，该日期相对于打包日期而不是相对于许可证的签发日期来表示。

示例用例：在自动打包过程中，对一系列视频使用带有此选项的单个Primetime DRM策略，将到期日期设置为相对于打包日期的30天。
