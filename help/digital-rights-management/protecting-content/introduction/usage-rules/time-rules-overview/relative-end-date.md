# 相对结束日期{#relative-end-date}

相对结束日期指定许可证到期日期，该日期表示为与包装日期相关的日期，而非与发放许可证的日期相关的日期。

示例用例：在自动打包流程中，对一系列视频使用具有此选项的单个Primetime DRM策略，将到期日期设置为与打包日期相对的30天。
