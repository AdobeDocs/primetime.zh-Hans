# DRM和运行时的最低安全级别{#minimum-security-level-for-drm-and-runtimes}

最低安全级别指定访问内容所需的安全级别。 您可以为每个组件指定单独的安全级别。

DRM/运行时模块的默认安全级别为10000。

示例用例：某些类型的内容（例如高清视频）可能比其他类型需要更高的安全级别。
