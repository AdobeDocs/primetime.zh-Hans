# DRM和执行时间的最低安全级别{#minimum-security-level-for-drm-and-runtimes}

最低安全级别指定访问内容所需的安全级别。 您可以为每个组件指定单独的安全级别。

DRM/运行时模块的默认安全级别为10000。

示例用例：某些类型的内容（例如HD视频）可能需要比其他类型更高的安全级别。
