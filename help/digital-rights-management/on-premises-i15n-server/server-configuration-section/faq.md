# 常见问题解答{#faq}

* ECI更改的发生频率是多久？
   * 每当新AdobeDRM客户端被发布时，都会添加ECI设备记录。

* ECI文件有多大？
   * 它们通常不到每台设备记录1 KB。

* 如果服务器缺少ECI设备记录，会发生什么情况？
   * 该特定类别的客户端将无法针对本地个性化服务器进行个性化设置，错误将记录到日志文件中。

* 如果服务器的CRL过期，会发生什么情况？
   * 服务器将停止正常工作，错误将记录到日志文件。
