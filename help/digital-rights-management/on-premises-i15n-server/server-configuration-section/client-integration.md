# 客户端集成{#client-integration}

为了引导客户端针对本地个性化服务器（与Adobe Hosted Global Indivialization Server不同）进行个性化，客户端应使用先前创建的本地DRM元数据。 由非个性化客户端使用特殊元数据执行许可证获取或初始化DRM，将导致客户端连接到自定义个性化服务器URL。

该文件夹中包含一个示例代码片 [!DNL client_sample] 段。
