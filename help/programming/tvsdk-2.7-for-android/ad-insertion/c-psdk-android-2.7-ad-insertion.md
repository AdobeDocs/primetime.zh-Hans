# 广告及其要求{#advertising-requirements}

您可以使用Adobe Primetime广告决策界面在VOD和实时/线性内容中插入广告。

Primetime广告决策与TVSDK合作，在您的视频流中识别广告机会、解析广告和插入已解决广告。

要在视频内容中加入广告，请确保广告和主要视频内容满足以下要求：

* 广告内容的HLS版本不能高于主内容的HLS版本。
* 无论主内容是否被复用，广告都不必复用（没有限制）。
