# 概述 {#configure-the-player-user-interface-overview}

通过TVSDK，您可以控制实时和视频点播(VOD)的基本播放体验。 TVSDK不为您配置播放器，而是在播放器实例上提供方法和属性，您可以使用这些方法和属性来配置播放器用户界面。
