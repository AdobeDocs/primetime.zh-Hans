# 概述 {#control-closed-caption-styling-overview}

您可以使用TextFormat类为隐藏式字幕轨道提供样式信息。 这将设置播放器显示的任何隐藏式字幕的样式。

此类封装隐藏式字幕样式信息，如字体类型、大小、颜色和背景不透明度。
