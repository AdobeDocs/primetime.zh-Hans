# 概述 {#requirements-overview}

浏览器TVSDK需要媒体内容、清单内容和软件版本的特定属性。

浏览器TVSDK版本包含以下播放器：

* `Primetime.js` 适用于桌面和移动设备上的HLS

播放器包括以下参考API：

* `Primetime.js` 浏览器TVSDK
* 用于核心播放的JavaScript API
* 参考演示播放器

      * Firefox 38+

      * Chrome 33+
