# 概述{#requirements-overview}

浏览器TVSDK需要媒体内容、清单内容和软件版本的特定属性。

浏览器TVSDK版本包括以下播放器：

* `Primetime.js` 桌面和移动设备上的HLS

播放器包括以下参考API:

* `Primetime.js` 浏览器TVSDK
* 用于核心回放的JavaScript API
* 参考演示播放器
