# 媒体播放和故障切换 {#media-playback-and-failover}

对于实时和视频点播(VOD)媒体，TVSDK通过下载与中分辨率比特率关联的播放列表来开始播放，并下载由该播放列表定义的媒体区段。 它快速选择高分辨率的比特率播放列表及其相关媒体并继续下载过程。
