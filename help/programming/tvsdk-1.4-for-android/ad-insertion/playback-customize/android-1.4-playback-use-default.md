# 使用默认播放行为 {#use-the-default-playback-behavior}

您可以选择使用默认广告行为。

要使用默认行为，请执行以下操作：

    *如果您实施自己的“AdvertisingFactory”类，请为“createAdPolicySelector”返回null 。
    
    *如果您没有“AdvertisingFactory”类的自定义实施，TVSDK将使用默认广告策略选择器。
