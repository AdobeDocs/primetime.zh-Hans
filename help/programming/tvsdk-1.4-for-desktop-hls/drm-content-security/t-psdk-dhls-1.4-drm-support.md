# 支持的DRM系统{#supported-drm-systems}

您可以使用PrimetimeDigital Rights Management(DRM)系统的功能提供对视频内容的安全访问。 或者，您也可以将第三方DRM解决方案用作Adobe集成Primetime DRM解决方案的替代方案。

请咨询您的Adobe代表，了解有关第三方DRM解决方案可用性的最新信息。
