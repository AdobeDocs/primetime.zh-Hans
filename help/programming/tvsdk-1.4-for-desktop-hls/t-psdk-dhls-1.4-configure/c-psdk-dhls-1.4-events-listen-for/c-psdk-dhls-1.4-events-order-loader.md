# 加载器事件{#loader-events}

您的播放器可以基于以下事件执行操作：

| 活动 | 意义 |
|---|---|
| `MediaPlayerItemLoader.completed` | 媒体资源加载成功完成。 |
| `MediaPlayerItemLoader.failed` | 加载媒体资源时出现问题。 |
