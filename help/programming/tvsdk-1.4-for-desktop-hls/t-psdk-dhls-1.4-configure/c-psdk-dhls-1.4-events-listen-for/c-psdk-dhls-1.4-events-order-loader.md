# 加载器事件{#loader-events}

您的播放器可以基于以下事件实施操作：

| 事件 | 意义 |
|---|---|
| `MediaPlayerItemLoader.completed` | 媒体资源加载已成功完成。 |
| `MediaPlayerItemLoader.failed` | 加载媒体资源时出现问题。 |
