# 概述 {#configure-the-player-user-interface-overview}

通过TVSDK，您可以控制实时和视频点播(VOD)的基本播放体验。 TVSDK提供了播放器实例上的方法和属性，您可以使用这些方法和属性来配置播放器用户界面，而不是为您配置播放器。
