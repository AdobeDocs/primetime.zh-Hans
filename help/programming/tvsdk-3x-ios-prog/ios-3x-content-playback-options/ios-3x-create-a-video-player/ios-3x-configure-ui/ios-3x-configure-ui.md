# 概述{#configure-the-player-user-interface-overview}

借助TVSDK，您可以控制实时和视频点播(VOD)的基本播放体验。 TVSDK在播放器实例上提供了可用于配置播放器用户界面的方法和属性，而不是为您配置播放器。
