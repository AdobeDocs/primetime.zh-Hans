# 预授权 {#preauthorize}

>[!NOTE]
>
>此页面上的内容仅供参考。 使用此API需要获得Adobe的当前许可证。 不允许未经授权使用。

预授权API可用于获取一个或多个资源的预授权决策，这样应用程序可以实施UI提示和/或内容过滤。

>[!IMPORTANT]
>
>授权API **必须** 在授予用户对指定资源的访问权限之前使用。

如果预授权API响应结果包含一个或多个具有拒绝的预授权决定的资源，则可以包括其他错误信息 **（请参阅下文附注）** 每个受影响的资源。

>[!IMPORTANT]
>
>增强的错误报告功能为拒绝的预授权决策添加其他错误信息，该功能可在请求时使用，因为必须在Adobe Primetime身份验证配置端启用它。

如果由于Adobe Primetime身份验证SDK错误或发生Adobe Primetime身份验证服务错误而无法提供预授权API请求，则附加错误信息（与上述配置无关）和资源将不会作为预授权API响应结果的一部分包含在内。

</br>

## `- (void) preauthorize:(nonnull PreauthorizeRequest *)request didCompleteWith:(nonnull AccessEnablerCallback<PreauthorizeResponse *> *)callback;`


**可用性：** v3.6.0+

**参数：**

- PreauthorizeRequest:用于传递API请求内容的请求对象；
- AccessEnablerCallback:用于返回API响应的回调对象；
- 预授权响应：用于返回API响应内容的响应对象；

 
</br>

    ///
