# 关于zipfiles文件夹 {#about-zipfiles}

此文件夹中名为zipfiles(https://git.corp.adobe.com/AdobeDocs/primetime.en/tree/main/zipfiles)的文件和文件夹是以前版本的XML格式文档，适用于以下区域：
* DRM(Digital Rights Management)
* TVSDK
* 动态Ad Insertion
* 参考实施
* Primetime编程
>[!NOTE]
>要访问这些文件，请下载文件夹并打开文件。
